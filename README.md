# Python Practice

Learning Python from scratch and writing automated tests with pytest.

## What's in here
- variables.py — storing values
- strings.py — text manipulation
- decisions.py — if/else logic
- lists.py — working with lists
- loops.py — for loops
- dicts.py — dictionaries and nested data
- functions.py — reusable functions
- mini_program.py — combining everything
- test_math.py — first pytest tests
- test_user.py — testing functions and dicts with pytest

## How to run the tests
pip install pytest
python -m pytest -v
